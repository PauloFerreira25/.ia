# Copy and Duplicate

When duplicating or copying content:
- Read the original source completely
- Understand its structure, patterns, and details
- Duplicate everything first, then modify what needs to change
- Never create from scratch when a copy was requested

When copying model X to create Y:
- Read model X in full
- Duplicate the structure exactly
- Update only what changes for Y
- Preserve patterns, style, and organization from X

Never:
- Create new when a copy was requested
- Assume structure without reading the original first
- Skip reading the original
